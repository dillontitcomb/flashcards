Click on a term to see its definition:

Operators

An operator is a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

Variables can be thought of as named containers. You can place data into these containers and then refer to the data simply by naming the container. Before you use a variable in a JavaScript program, you must declare it. Variables are declared with the var keyword.

Variable Naming Conventions

Variable naming conventions are used to avoid declaring variables as global variables(i.e: var result = instead of result =)

Functions

Functions are blocks of code designed to perform a particular task. They are executed when "something" invokes it.

Methods

Methods are the actions that can ne performed on objects. A JavaScript method is a property containing a function definition.

Arguments

Arguments are built into the function, the argument object contains an array of the arguments used when the function was called(invoked).

Parameters

Parameters are the variables in a method definition. When a method is called, the arguments are the data that you pass into the method's parameters.

Return

The return statement stops the execution of a function and returns a value from that function.

Chaining Methods

Chaining Methods refers to repeatedly calling one method after another on an object, in one continuous line of code. (ex: $("myClass").show().toggle();)

Strings

Strings are used simply to store a series of characters in a row, like "John Doe". A string can be any text inside quotes.

Booleans

Booleans represent two values, either "True", or "False". If value parameter is omitted or is 0, null, false, NaN, undefined, or the empty string(""), the object has an initial value of false.

Undefined

Undefined or null are both used to represent the absence of some value.

NaN

NaN or Not-A-Number is used to indicate that a value is not a legal number.

Escape

The escape function is used to make a string portable, so that it can ne transmitted across any network to any computer that supports ASCII characters. (deprecated in JS version 1.5)

Alert

Alert is a method that will pop up a window on the page with a customizable message for the user.

Comments

Comments are used to add additional information to your code to help people unfamiliar with it identify important and or ticky elements.

Jquery

Jquery is a concise and fast JavaScript library that is used to simplify event handling among other things for speedy website development.

Attributes

Attributes set or return attributes and values of the selected elements.
